# CMAME-D-26-01111 Article Package

This directory contains article-specific material implementations, representative simulations, and benchmark records for the CMAME article.

## Contents

- `single_element/`: thermomechanically coupled PA6 single-element example and the `TVPkinIFT_heat` reference material implementation.
- `noniso_dogbone/`: representative non-isothermal dogbone setup.
- `gpu_benchmark/`: archived CPU/GPU benchmark logs used to document the runtime scaling behavior reported for the article.
- `INPUT_FILES.md`: summary of article-specific CSV parameter tables, mesh files, and generated-file exclusions.

Generated ADIOS `.bp` result directories are not included.
